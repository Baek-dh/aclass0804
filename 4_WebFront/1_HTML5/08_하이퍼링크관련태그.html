하이퍼링크(하이퍼 텍스트) 관련 태그

        Hyperlink(Hypertest) : 다른 문서로 즉시 접근(이동)하는 텍스트

        HTTP(HyperText Transfer Protocol) : 하이퍼텍스트로 이루어진 문서를 전송하는 규약
                                            -> HTML

a(Anchor) 태그

        - a태그는 현재 HTML 문서와 
        다른 HTML 문서를 연결해서 
        이동할 수 있게하는 기능을 제공하는 태그.
        (페이지 이동)

        - a태그로 감싸진 문자열 또는 이미지 == 하이퍼링크

        - a태그는 inline 형식의

        
        ** 하이퍼링크는 웹 문서가 다른 문서와 구분될 수 있는 핵심 기능.
            1) 다른 페이지로 이동
            2) 현재 페이지 내에서 이동

href 속성 : 참조할 페이지 주소(이동할 페이지 주소)를 작성하는 속성

target 속성 : 연결된 문서를 어디에 열 것인지 지정하는 속성
