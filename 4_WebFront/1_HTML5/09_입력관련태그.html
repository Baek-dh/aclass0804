        ** text 관련 type의 공통 속성 **

다중 선택 관련 input 태그

        1) type="checkbox"
            - 여러 값 중 원하는 값을 모두 체크(선택)

        2) type="radio"
            - 여러 값 중 단 하나만 선택 가능

label 태그

       - input 태그 제목으로 사용되는 태그

       - 사용 방법
       1) input 태그에 id 속성 값을 작성
       2) 원하는 위치에 label 태그를 작성한 후
          for 속성을 작성.
          이 때 for 속성의 값은 
          1)에서 작성한 id 속성 값과 같아야 한다.

form 태그

        - 내부에 작성된 input 태그의 값을 서버로 전달
         -> 이 때 전달되는 값의 형식은 k:v 형태

         key == input 태그의 name 속성 값

         value == input 태그에 작성된 값
                  또는 value 속성 값


        - form 태그의 속성
            1) action(중요!) : 내부에 작성된 input 값을 어디로 전달할지 지정하는 속성
                        -> 주소를 작성하게 됨.
            
            2) method(방식) : 데이터를 서버로 전달할 때의 방식을 지정하는 속성 

            3) name : form 태그의 이름 지정(JS 사용)

            4) target : action에 지정된 주소를 새 탭/현재 탭 중 어디서 열지 지정
