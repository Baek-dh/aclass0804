FlexBox

        요소의 정렬되는 방향, 순서, 요소 간의 간격을
        개발자가 수치적으로 처리하지 않아도
        알아서 유연하게 처리해주는 형식.

        * FlexBox 이용 시 반드시 알아야 할 것!

        1. FlexBox의 구성
        - flex-container : 정렬이 필요한 요소를 감싸는 요소 
        - item : 정렬이 필요한 요소(정렬 적용 대상)
        
        -> flex 관련 css 속성이 flex-container, item 구문으로 나누어져 있음.


        2. FlexBox의 축(axis)
        - 중심축(main axis)
        - 교차축, 반대축(cross axis)
