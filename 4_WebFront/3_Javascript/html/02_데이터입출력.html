        - 내용을 읽어올 때 태그+속성+내용을 모두 포함함.

        - 내용을 변경할 때 태그는 HTML 요소로 해석됨. (HTML 해석 O)

        - alert(알리다, 경보, 경고)
        - 브라우저에 대화상자(알림창)를 띄우는 함수

window.confirm("내용")

        질문에 대한 "예" / "아니오" 결과를 얻고자 할 때 사용하는 대화 상자
        - 예/확인     : true 반환
        - 아니오/취소 : false 반환

window.prompt("내용")

        텍스트를 작성할 수 있는 대화 상자

        - 확인 : 입력한 값을 반환
        - 취소 : null 반환
